Setup Guide: Intelligent Video Recorder System

This guide will walk you through the steps to set up and run the Intelligent Video Recorder System application. By the end of this guide, you will have the application running on your local machine.

Note: This application requires significant computational resources due to real-time video processing and object detection. To ensure smooth performance and avoid system crashes, please make sure your machine meets the following requirements:

  • A modern multi-core CPU (Intel i5 or higher, or equivalent).
  • RAM 16GB is recommended.
  • A dedicated GPU (e.g., NVIDIA GTX 1060 or higher) for faster processing, especially for object detection tasks.

Running this application on underpowered hardware may result in slow performance or system instability.

Step 1: Install Python

If you don't already have Python installed, download and install it from the official website:

  • Go to python.org/downloads.
  • Download the latest version of Python (3.8 or higher is recommended).
  • Run the installer and make sure to check the box that says Add Python to PATH during installation.

To verify the installation, open a terminal or command prompt and run:

python --version

You should see the installed Python version.

Step 2: Create a Virtual Environment

A virtual environment is a self-contained directory that contains a Python installation and all the dependencies for your project. This avoids conflicts with other Python projects.

To create a virtual environment:

python -m venv video_recorder_env

Activate the virtual environment:

  • On Windows:
    • video_recorder_env\Scripts\activate
  • On macOS/Linux:
    • source video_recorder_env/bin/activate

You should see the virtual environment name in your terminal prompt, e.g., (video_recorder_env).

Step 3: Install Requirements

The application requires several Python packages to run. These are listed in a requirements.txt file.

Download the requirements.txt file from the project repository

Install the requirements by running:

pip install -r requirements.txt

This will install all the necessary dependencies.

Step 4: Download the Application Code

Download the application code from the repository or copy the Python script (main.py) to your project directory.

Your project directory should look like this:

my_project/
├── main.py
├── requirements.txt
└── video_recorder_env/

Step 5: Run the Streamlit Application

Once everything is set up, you can run the application using Streamlit.

In your terminal or command prompt, navigate to the project directory and run:

streamlit run main.py

This will start the Streamlit server and open the application in your default web browser.

You should see the application interface with options to choose between upload and camera modes.

Step 6: Using the Application

Follow these steps to use the application:

  1. Choose Recording Method: Select either upload or camera mode.
  2. For Camera Mode: Enter the IP camera address or use 0 for the default webcam. Click Start Streaming to begin.
  3. For Upload Mode: Upload a video file (mp4 or avi). Adjust the confidence and IoU thresholds as needed.
  4. Stop Recording: Click the Stop Recording button to end the recording.

The recorded videos will be saved in the recordings directory.

Note:

If you encounter any issues, make sure:

  • Your virtual environment is activated.
  • All dependencies are installed correctly.
  • You have the correct permissions to access the camera or video file.

Conclusion

Congratulations! You have successfully set up and run the Intelligent Video Recorder System application. You can now use it to detect motion, record videos, and analyze objects in real-time.
